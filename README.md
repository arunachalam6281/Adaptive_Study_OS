# 🚀 Adaptive Study OS

Adaptive Study OS is a full-stack productivity and coding analytics platform designed to help students improve study consistency, track coding practice, and manage learning activities from a single application.

The platform combines a Chrome Extension, Spring Boot backend, MySQL database, and React Native mobile application to provide real-time coding activity tracking and productivity insights.

---

## 📖 Overview

Adaptive Study OS enables users to:

* Track LeetCode submissions in real time
* Monitor coding consistency and submission history
* Manage study tasks and daily goals
* Analyze productivity patterns and learning progress
* Access centralized learning insights through a mobile dashboard

## ⚙️ Installation & Setup

```bash
git clone https://github.com/arunachalam6281/Mini-Project.git

```

---

## 🔹 Backend Setup

Navigate to backend folder:

```bash
cd backend
```

Install dependencies and run:

```bash
mvn clean install
mvn spring-boot:run
```

Backend will run at:

```text
http://localhost:8080
```

---

## 📱 Frontend Setup

Navigate to frontend folder:

```bash
cd frontend/ToDoApp
```

Install dependencies:

```bash
npm install
```

Start the application:

```bash
npx expo start
```

Run using:

* Expo Go App
* Android Emulator
* iOS Simulator

---

## 🌐 Chrome Extension Setup

1. Open Google Chrome
2. Navigate to:

```text
chrome://extensions/
```

3. Enable **Developer Mode**
4. Click **Load unpacked**
5. Select the `extension` folder

The extension will start monitoring LeetCode submissions automatically.
